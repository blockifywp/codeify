== Description ==

Codeify is an ultra-sleek, customizable WordPress block theme engineered for diverse industries, offering a fully responsive design, top-tier performance, and seamless user experience to propel your digital presence to new heights. It showcases unique effects which are sure to impress your visitors. The theme is perfect for businesses, startups, AI companies and creative agencies.

== Installation ==

1. In your site's admin panel, go to Appearance > Themes and click `Add New`.
2. Type "Codeify" in the search field.
3. Click `Install` and then `Activate` to start using the theme.
4. Navigate to Appearance > Editor.
5. Edit the page content as you wish, you can add, remove or customize any of the blocks.
6. Enjoy :)
